Документація до кастомних HTML-тегів

1. <highlight>
Опис:
Виділяє текст яскравим кольором для акцентування уваги на важливій інформації.

Атрибути:

color — задає колір підсвічування (наприклад, "yellow", "#ff0").
Приклад:

html


<highlight color="yellow">Увага! Це важлива інформація.</highlight>
2. <note>
Опис:
Відображає коротку примітку або підказку для користувача у стилізованому блоці.

Атрибути:

type — тип нотатки: "info", "tip", "reminder".
Приклад:

html


<note type="tip">Збережіть зміни перед виходом.</note>
3. <warning>
Опис:
Виводить попередження з іконкою та кольоровим фоном для критичних повідомлень.

Атрибути:

level — рівень небезпеки: "low", "medium", "high".
Приклад:

html


<warning level="high">Небезпека! Не вимикайте пристрій.</warning>
4. <spoiler>
Опис:
Ховає текст, який можна розкрити за бажанням користувача (наприклад, для спойлерів у статтях).

Атрибути:

label — текст кнопки для розкриття.
Приклад:

html


<spoiler label="Показати відповідь">Відповідь: 42</spoiler>
5. <timeline>
Опис:
Створює шкалу часу для відображення послідовності подій.

Атрибути:

orientation — "vertical" або "horizontal".
Приклад:

html


<timeline orientation="vertical">
  <event date="2020">Початок проекту</event>
  <event date="2021">Запуск MVP</event>
</timeline>
6. <avatar>
Опис:
Відображає аватар користувача з можливістю завантаження або зміни фото.

Атрибути:

src — шлях до зображення.
size — розмір аватара (наприклад, "small", "large").
Приклад:

html


<avatar src="user.jpg" size="large"></avatar>
7. <rating>
Опис:
Додає рейтинг у вигляді зірочок або інших іконок для оцінки товару чи послуги.

Атрибути:

max — максимальна кількість зірок.
value — поточний рейтинг.
Приклад:

html


<rating max="5" value="4"></rating>
8. <carousel>
Опис:
Створює слайдер для зображень або іншого контенту з можливістю прокручування.

Атрибути:

autoplay — автоматичне прокручування (true/false).
interval — інтервал у мілісекундах.
Приклад:

html


<carousel autoplay="true" interval="3000">
  <img src="1.jpg">
  <img src="2.jpg">
</carousel>
9. <badge>
Опис:
Відображає бейдж або лейбл для позначення статусу, категорії чи кількості.

Атрибути:

color — колір бейджа.
text — текст бейджа.
Приклад:

html


<badge color="green" text="Новий"></badge>
10. <progress-ring>
Опис:
Кільцевий індикатор прогресу для відображення виконання задачі.

Атрибути:

value — поточне значення.
max — максимальне значення.
Приклад:

html


<progress-ring value="70" max="100"></progress-ring>
11. <code-block>
Опис:
Форматує та підсвічує блок коду для зручного читання.

Атрибути:

language — мова програмування для підсвічування.
Приклад:

html


<code-block language="javascript">
  console.log('Hello, world!');
</code-block>
12. <quote>
Опис:
Відображає цитату з іменем автора у стилізованому блоці.

Атрибути:

author — ім'я автора цитати.
Приклад:

html


<quote author="Альберт Ейнштейн">Уява важливіша за знання.</quote>
13. <stepper>
Опис:
Відображає покрокову інструкцію або процес із виділенням поточного кроку.

Атрибути:

current — номер поточного кроку.
Приклад:

html



<stepper current="2">
  <step>Реєстрація</step>
  <step>Підтвердження email</step>
  <step>Завершення</step>
</stepper>
14. <emoji>
Опис:
Вставляє анімований емодзі для емоційного забарвлення тексту.

Атрибути:

type — тип емодзі (наприклад, "smile", "heart").
Приклад:

html


<emoji type="smile"></emoji>
15. <divider>
Опис:
Додає стилізований роздільник між секціями сторінки.

Атрибути:

style — стиль лінії ("dashed", "solid", "double").
Приклад:

html


<divider style="dashed"></divider>
16. <map-pin>
Опис:
Відображає інтерактивну мітку на карті з підказкою.

Атрибути:

lat — широта.
lng — довгота.
label — підпис мітки.
Приклад:

html


<map-pin lat="50.4501" lng="30.5234" label="Київ"></map-pin>
17. <faq>
Опис:
Створює блок із питаннями та відповідями, які можна розгортати.

Атрибути:

немає (структура визначається вкладеними тегами).
Приклад:

html


<faq>
  <question>Як зареєструватися?</question>
  <answer>Натисніть кнопку "Реєстрація" у верхньому меню.</answer>
</faq>
18. <sound>
Опис:
Відтворює короткий аудіофрагмент при натисканні або автоматично.

Атрибути:

src — шлях до аудіофайлу.
autoplay — автоматичне відтворення (true/false).
Приклад:

html


<sound src="click.mp3" autoplay="false"></sound>
19. <theme-switch>
Опис:
Додає перемикач між світлою та темною темами інтерфейсу.

Атрибути:

немає.
Приклад:

html


<theme-switch></theme-switch>
20. <confetti>
Опис:
Анімує святковий ефект конфетті на сторінці для святкування подій.

Атрибути:

duration — тривалість анімації у мілісекундах.
colors — список кольорів через кому.
Приклад:

html


<confetti duration="2000" colors="red,green,blue"></confetti>
